<h1>Depot</h1>

<p>Depot is a web-based shopping cart application based on book "Agile Web Development with Rails 5" by Sam Ruby. </p>
<p>Use cases:<br>
  Buyer - the buyer uses depot to browse the products we sell, select some to purchase, and supply the information needed to create an order.<br>
  Seller - the seller uses Depot to maintain a list of products to sell, to determine the orders that are awaiting shipment, and to mark orders as shipped. </p>
